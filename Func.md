## 🌌 [Project] 전북과학고 실시간 천문 관측 대시보드

### 1. UI/UX 레이아웃 설계
전체 화면은 **스텔라리움(Stellarium) 방식의 동적 배경**을 풀스크린으로 활용하며, 각 기능(Func)은 사용자 인터랙션에 따라 유연하게 움직이는 대시보드 형태로 구성함.

| 기능 블록 | 위치 | 인터랙션 (Click Toggle) |
| :--- | :--- | :--- |
| **Func 1: 관측 정보** | 좌측 하단 | **미니멀 모드**: 관측 가능 여부/현재 시각만 표시 <br> **확장 모드**: 상세 주소 및 천문 현상 리스트 출력 |
| **Func 2: 예보 정보** | 중앙 하단 | **미니멀 모드**: 현재 기온/강수 확률 요약 <br> **확장 모드**: 24시간 타임라인형 기상 그래프 전개 |
| **Func 3: 외부 링크** | 우측 하단 | **미니멀 모드**: 아이콘 형태의 바로가기 <br> **확장 모드**: 텍스트 포함 상세 링크 리스트 노출 |

### 2. 상세 기능 명세 (Functional Requirements)

#### **Func 1. 관측 데이터 대시보드 (좌측 하단)**
*   **Location**: 전북특별자치도 익산시 금마면 아리랑로 124-46 (전북과학고등학교 천문대)
*   **Weather**: OpenWeatherMap API 연동 실시간 기온, 습도, 풍향 정보.
*   **Status**: 운량(Cloudiness)과 시정(Visibility) 데이터를 분석하여 '최적/보통/불가' 3단계 표시.
*   **Solar/Lunar**: 일출·일몰 시각 및 달의 위상(Moon Phase) SVG 애니메이션 시각화.
*   **Time Module**: 일반 KST 시각과 천문 계산에 필요한 율리우스일(JD) 또는 항성시(LST) 병기.

#### **Func 2. 시간별 기상 타임라인 (중앙)**
*   **Timeline**: 향후 24시간의 기상 변화를 가로 스크롤 차트로 구현.
*   **Alert System**: 비, 눈, 강풍 등 관측 방해 요소 발생 시 **Red Flash** 효과 또는 경고 아이콘 강조.
*   **Data Source**: 기상청 단기예보(JSON) 데이터를 파싱하여 실시간 업데이트.

#### **Func 3. 리소스 게이트웨이 (우측 하단)**
*   **Link Group**: 외부 사이트 이동 시 현재 세션을 유지하며 새 탭으로 연결(target="_blank").
*   **Design**: 각 기관의 로고를 활용한 세련된 카드 뉴스 형태의 UI 채택.

#### **Func 4. 인터랙티브 천구 배경 (Main Canvas)**
*   **Engine**: **Three.js** 또는 **D3-celestial** 라이브러리 활용.
*   **Mechanism**: 사용자의 마우스 드래그 또는 자이로 센서(모바일)에 반응하여 방위각(Azimuth)과 고도(Altitude)에 따른 별자리 위치 실시간 계산.
*   **Real-time**: 전북과학고의 위경도 값을 기반으로 현재 시각 하늘에 떠 있는 실제 천체 배치(Catalog 데이터) 동기화.

### 3. 기술 스택 및 구현 전략

*   **Frontend**: **Next.js 14** (App Router 활용으로 빠른 상태 전환 구현)
*   **State Management**: 토글 상태(Expanded/Collapsed) 및 실시간 시간 데이터 관리를 위한 **Zustand** 사용.
*   **Animation**: 카드 UI의 확장/축소 애니메이션을 위한 **Framer Motion** 도입.
*   **Celestial Logic**: 별의 위치 계산을 위한 **Astronomy.js** 또는 **Hypatia** 라이브러리 연동.

이 기획안은 전북과학고의 전문성과 최첨단 웹 기술이 결합된 형태임. 사용자가 접속하자마자 마치 학교 천문대 돔 안에 있는 듯한 몰입감을 느끼도록 설계하는 것이 개발의 핵심임.
