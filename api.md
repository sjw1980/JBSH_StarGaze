현재 앱에서 실시간 데이터가 필요한 항목은 총 **7가지**입니다:

---

### 현재 실시간 데이터 항목

| # | 항목 | 현재 구현 방식 | 갱신 주기 |
|---|------|--------------|---------|
| 1 | **현재 날씨** (기온·체감·습도·풍향·풍속·운량·시정·강수) | OpenWeatherMap API (`/api/weather`) | 5분 |
| 2 | **24시간 기상 예보** (기온·운량·강수확률 차트) | OpenWeatherMap Forecast API (`/api/forecast`) | 30분 |
| 3 | **일출/일몰 시각** | OpenWeatherMap 응답 내 `sunrise`/`sunset` 포함 (위 1번과 동일) | 날씨 갱신 시 |
| 4 | **달 위상** (위상각, 이름) | `astronomy-engine` 라이브러리로 로컬 계산 | 앱 로드 시 1회 |
| 5 | **KST 현재 시각** | `Date()` 로컬 계산 | 1초 |
| 6 | **항성시 (LST)** | 율리우스일 + 관측소 경도로 로컬 계산 | 1초 |
| 7 | **태양 위치** (천구에서의 RA/Dec) | VSOP87 근사식으로 로컬 계산 (`StarCanvas`) | 앱 로드 시 1회 |

---

**외부 API 의존 항목**: 1·2·3번 (OpenWeatherMap — API 키 없으면 데모 데이터로 대체)  
**로컬 계산 항목**: 4·5·6·7번 (네트워크 불필요, 수식으로 실시간 계산)
